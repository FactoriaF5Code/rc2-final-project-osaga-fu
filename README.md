## Installation

### FRONTEND

En la carpeta frontend introduce los siguientes comandos:

```
npm install
npm intall react-router-dom
npm install axios
```

Para lanzar el fronted:

```
npm run dev
```

### BACKEND 

Para lanzar el backend:

```
./mvnw spring:boot run
```
